Download only audio of video (Add -4 to the end if there are any errors):
yt-dlp -f bestaudio -x --audio-format wav URL

Download a playlist:
yt-dlp -f bestaudio -x --audio-format wav -o "%(title)s.%(ext)s" PLAYLIST_URL

List available formats:
yt-dlp -F URL

Download best quality video with audio:
yt-dlp -f bestvideo+bestaudio/best URL

Download video and audio from certain time to another time:
yt-dlp -f bestvideo+bestaudio/best --download-sections "*1:10-1:40" URL

Download audio from certain time to another time:
yt-dlp -f bestaudio/best --download-sections "*1:10-1:40" URL

If getting 403 Forbidden error with yt-dlp, run this command:
yt-dlp --rm-cache-dir
